UNEMPLOYMENT ANALYSIS WITH PYTHON
================================

PROJECT OVERVIEW
----------------
This project analyzes unemployment rate data in India using Python.
The goal is to understand trends, identify the impact of Covid-19,
and extract insights useful for economic and social policy planning.

DATASET
-------
Source: Unemployment in India Dataset
Features include:
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Labour Participation Rate (%)

OBJECTIVES
----------
- Clean and preprocess unemployment data
- Analyze unemployment trends over time
- Study the impact of Covid-19 on unemployment
- Identify key patterns and changes
- Visualize trends using graphs

TECHNOLOGIES USED
-----------------
- Python
- Pandas
- Matplotlib

PROJECT STRUCTURE
-----------------
unemployment-analysis/
│
├── data/
│   └── Unemployment_in_India.csv
│
├── src/
│   └── unemployment_analysis.py
│
├── README.txt
└── requirements.txt

HOW TO RUN
----------
1. Install dependencies:
   pip install -r requirements.txt

2. Run the analysis:
   python src/unemployment_analysis.py

KEY INSIGHTS
------------
- Unemployment sharply increased during the Covid-19 period (2020).
- Post-pandemic data shows gradual recovery.
- Visual trends help identify economic stress periods.
